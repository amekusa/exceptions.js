## Installation
```sh
npm i @amekusa/exceptions
```

In your JS:
```js
import { Exception } from '@amekusa/exceptions';      // ESM
const { Exception } = require('@amekusa/exceptions'); // CJS
```

### Available Exceptions:
```js
import {
  Exception,
  NoSuchProp,
  InvalidType,
  InvalidValue
} from '@amekusa/exceptions';
```
